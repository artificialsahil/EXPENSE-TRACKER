# Expense Tracker in C

A simple command-line *Expense Tracker* built in C to manage and monitor daily expenses. Users can add expenses, view all entries, calculate total spending, and save data to a file. This project demonstrates core C concepts such as *structures, arrays, loops, and file handling*.

## Features

- Add expense with name and amount
- View all recorded expenses
- Calculate total expenses
- Save expenses to expenses.txt for future reference
- Simple menu-driven interface

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/expense-tracker-c.git

2. Navigate to the project directory:

cd expense-tracker-c


3. Compile the program using GCC:

gcc expense_tracker.c -o expense_tracker


4. Run the program:

./expense_tracker



Usage

1. Run the program.


2. Select an option from the menu:

Add Expense: Enter name and amount.

View Expenses: Display all recorded expenses.

Total Amount: Show total spending.

Save to File: Save current expenses to expenses.txt.

Exit: Close the program.



3. Follow on-screen prompts to manage your expenses.
